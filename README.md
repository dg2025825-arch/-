## 🖥️ 페이지 구성

### 📊 Page 1 — 독감 발생률 전체 분석 (`app.py`)
독감 데이터만을 활용하여 계절별 발생 패턴을 다각도로 분석합니다.

| 섹션 | 내용 |
|------|------|
| 핵심 통계 요약 | 전체 최고 ILI, 최고 발생 시즌, 겨울/여름 평균 ILI |
| 시즌별 ILI 주간 추이 | 시즌 선택 가능한 인터랙티브 라인 차트, 유행 기준선(6.6) 표시 |
| 계절별 평균 ILI 비교 | 막대 그래프(± 표준편차) + 박스플롯 |
| 시즌 × 월별 히트맵 | 시즌과 월을 교차한 평균 ILI 히트맵 |
| 시즌별 최고 ILI & 유행 지속 기간 | 막대 그래프 + 요약 데이터 테이블 |
| 종합 분석 및 평가 | 주요 발견 사항, 데이터 유의사항, 결론 |

---

### 🌡️ Page 2 — 날씨와 독감 상관관계 (`pages/2_날씨_독감_상관관계.py`)
독감 데이터와 기온 데이터를 결합하여 날씨와 독감 발생률의 통계적 관계를 분석합니다.

| 섹션 | 내용 |
|------|------|
| 핵심 상관관계 지표 | 피어슨 r, 스피어만 ρ, p-value, 관계 강도 |
| 평균기온 vs ILI 산점도 | 계절별 색상 구분 + 전체 추세선 |
| 월별 이중 축 비교 | 평균기온(막대) + 평균 ILI(라인) 이중 축 차트 |
| 계절별 세부 상관관계 | 계절별 피어슨 r 표 + 4분할 산점도 |
| 기온 구간별 평균 ILI | 기온 구간(영하 ~ 30℃ 이상)별 ILI 막대 그래프 |
| 특정 시즌 상세 비교 | 시즌 선택 시 해당 시즌의 월별 기온-ILI 이중 축 비교 |
| 종합 평가 | 통계 결과 요약, 과학적 배경, 한계, 결론 |
